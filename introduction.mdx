# EduLearn-Hub Pi Day

EduLearn-Hub Pi Day is a full-stack educational web application that celebrates Pi Day (March 14) through immersive, interactive experiences. Built with React, TypeScript, and a futuristic dark aesthetic, it makes learning about π engaging for everyone.

## What is EduLearn-Hub Pi Day?

Pi Day is an annual celebration on March 14 (3/14), matching the first three digits of π = 3.14159... This project turns that celebration into a rich learning experience with:

## Tech stack

The project is a **pnpm monorepo** combining a React frontend with an Express backend:

| Layer | Technology |
|---|---|
| Frontend framework | React 18 + TypeScript |
| Routing | Wouter |
| Animations | Framer Motion |
| UI components | Radix UI + Tailwind CSS |
| Build tool | Vite |
| API server | Express 5 |
| Database | PostgreSQL + Drizzle ORM |
| Validation | Zod v4 |
| API spec | OpenAPI 3.1 + Orval codegen |
| Monorepo | pnpm workspaces |

## Repository structure

```
EduLearn-Hub/
├── artifacts/
│   ├── api-server/        # Express 5 API server
│   └── pi-day/            # React SPA (the main Pi Day app)
├── lib/
│   ├── api-spec/          # OpenAPI spec + Orval codegen config
│   ├── api-client-react/  # Generated React Query hooks
│   ├── api-zod/           # Generated Zod schemas
│   └── db/                # Drizzle ORM schema + DB connection
└── scripts/               # Utility TypeScript scripts
```

<Note>
  The Pi Day frontend (`artifacts/pi-day`) is a self-contained React SPA that currently runs independently of the backend. The API server provides a foundation for future backend-connected features.
</Note>
